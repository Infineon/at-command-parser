AT Command Parser API

General Description

The AT Command Parser library creates a worker thread for processing input data.

The priority of the thread is CY_RTOS_PRIORITY_NORMAL. The macro CY_RTOS_PRIORITY_NORMAL is defined in abstraction-rtos/include/COMPONENT_THREADX/cyabs_rtos_impl.h.
