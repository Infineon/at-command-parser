typedef at_cmd_msg_base_t *(* at_cmd_parser_callback_t) (uint32_t cmd_id, uint32_t serial, uint32_t cmd_args_len, uint8_t *cmd_args)

AT Command Parser callback function prototype.

The callback parses the command arguments and allocates a command message structure. The structure is defined by the application but must begin with at_cmd_msg_base_t as the first element of the structure.

Parameters
[in]cmd_id: Command id of the command
[in]serial: Serial number of the command
[in]cmd_args_len: Length of the argument command string
[in]cmd_args: Argument command string,
Returns
Pointer to allocated message structure or NULL

◆ at_cmd_transport_is_data_ready

typedef bool(* at_cmd_transport_is_data_ready) (void *opaque)

Transport layer is data ready function prototype.

Routine is called by the AT Command Parser library to query whether input is available to be read.

NOTE: If the application would like the library reader thread to block until data is available rather than poll, this routine should not return until data is available.

Parameters
[in]opaque: Optional opaque pointer passed to the library during initialization.
Returns
true if data is available for reading, false otherwise.
